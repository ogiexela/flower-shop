
## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Documentation

http://localhost:3000/api

http://localhost:3000/index.html


## Test

```bash
# unit tests
$ yarn test

# test coverage
$ yarn test:cov
```
